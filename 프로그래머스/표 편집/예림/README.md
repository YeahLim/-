# 표 편집

## :timer_clock: **Time**
1시간 40분

## :pushpin: **Algorithm**
더블 링크드 리스트

## :round_pushpin: **Logic**
1. 현재 선택한 행 : Node 객체에 prev, next를 이용해 현재 위치 파악
2. 삭제 & 복구 : stack 이용, Node객체에 boolean deleted 필드 추가 


## :black_nib: **Review**
- `LinkedList`는 탐색시간이 오래 걸린다. 탐색 시간이 짧으면서, 삽입과 삭제를 해야하는 경우, `Double LinkedList`를 이용하자
