## :round_pushpin: **Logic**

1. 알고리즘

   주어진 삼각형을 직각 삼각형으로 변형한뒤 점화식을 생성하면 다음과 같다.
   
    ```
    점화식
    (n번째의 최대값) = max((n-1번째의 현재 열-1의 최대값),(n-1번째의 현재 열 최대값)) + (n번째의 값)
    ```
2. 전체 로직
  ```
    맨 첫번째 값 초기화

    for(2번째 행 -> n번째 행){
      for(첫번째 열 -> 마지막 열){
         이전 왼쪽 열 최대값 구하기
         이전 현재 열 최대값 구하기

         현재 열 최대값 = max(이전 왼쪽 열 최대값 , 이전 현재 열 최대값) + 현재 열 값
      }
    }

   가장 마지막 행에서 가장 큰 값 출력
  ```
